# ✨ Velora Music

**Velora Music** is a modern and lightweight music player built for Windows.  
It is designed to provide a **smooth and fast music experience** with a clean interface and minimal system usage.

Velora Music focuses on **simplicity, speed, and usability** so you can enjoy your music without distractions.

---

# 🚀 Features

🎧 **Modern UI**  
Clean and simple interface designed for comfortable music listening.

⚡ **Fast Performance**  
Starts instantly and plays music smoothly without heavy CPU usage.

📂 **Open With Velora Music**  
Double-click any supported audio file and it will open directly in Velora Music.

🎵 **Multiple Audio Format Support**

- MP3
- WAV
- FLAC
- M4A

📺 **Now Playing Screen**  
Displays the currently playing song with playback controls.

🖥 **Windows Integration**

- Desktop shortcut
- Start menu entry
- File association support

💾 **Lightweight Installer**  
Quick installation with minimal storage usage.
